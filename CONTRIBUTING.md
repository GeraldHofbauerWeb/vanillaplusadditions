# Contributing to VanillaPlusAdditions

Thank you for your interest in contributing to VanillaPlusAdditions! We welcome contributions from the community to help make this mod even better.

## Ways to Contribute
- **Bug Reports**: If you find a bug, please open an issue on GitHub with details on how to reproduce it.
- **Feature Requests**: Have an idea for a new feature? Open an issue to discuss it.
- **Pull Requests**: We accept pull requests for bug fixes, new features, and documentation improvements.

## Development Setup
See the [Development section in README.md](README.md#development) for instructions on how to set up your development environment.

## Coding Standards
- Follow the existing code style and naming conventions.
- Use `AbstractModule` and `AbstractModuleConfig` for new features where possible.
- Ensure your code is well-commented, especially for complex logic.
- Add debug logging where appropriate using the module's logger.

## AI Usage Guidelines
This project is developed with AI assistance. If you use AI to help with your contribution, please:
- Review all AI-generated code carefully for bugs and security issues.
- Be transparent about AI usage in your pull request description.
- Ensure the final result meets the project's quality standards.
